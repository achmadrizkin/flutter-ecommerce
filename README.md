# ✔️ Flutter E-Commerce App

A Flutter app based on the design of the Flutter Music App App, created by *Achmad Rizki Nur Fauzie*,
No source code, because all of my data is not my property

## Features
- Responsive Apps
- Authentication using google
- CRUD from API
- Splash Screen Animation
- Add Cart Product
- Search product
- Chatting
- Check if phone was connected to internet
- Payment Gateway (Razorpay)
- Handle back button
- Validation (cannot buy your own product, cannot chat yourself, cannot add cart your own product)
- Your product, cart, transaction (its dynamic data, u only can see your own)
- and etc.
